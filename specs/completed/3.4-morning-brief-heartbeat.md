## What We're Building

Two cron-driven surfaces:

**Morning Brief** (7 AM daily) — a rich, personalized brief delivered via
Telegram, posted to the Pages UI, and written to Obsidian. 7 sections assembled
from live data: today's calendar, pending approvals, memory highlights, one open
loop, one gift (unexpected connection from last night's cron), top news headlines,
and a Bible verse. Feels like a thoughtful chief of staff who read everything
while you slept.

**Predictive Heartbeat** (every 30 minutes, 8 AM–8 PM) — a lightweight
background check that surfaces when something needs attention. Open loop
detection (started something but no completion signal in expected timeframe),
metric divergence (pattern deviating from baseline), pending action expiry
warnings. Delivers via Telegram only when something is actionable — not
every 30 minutes unconditionally.

**Weekly Synthesis** (Friday 5 PM) — a longer weekly reflection. Patterns across
the week, themes across domains, one significant prediction for the week ahead.
Delivered via Telegram + Obsidian only (not Pages UI notification).

When complete: Matt wakes up, gets a Telegram message from THE Brain with his
day at a glance. Throughout the day, THE Brain pings him only when something
genuinely needs attention. Friday afternoon, a weekly reflection arrives.

## THE Brain Laws Check

### Law 1: One Public Face
No new HTTP surface. Three new cron triggers added to `scheduled()` dispatch.
Telegram delivery is outbound-only from Workers — no new inbound route.

### Law 2: Zero-Knowledge
Brief assembly uses KEK-based recall (same pattern as nightly cron). Memory
content decrypted in Worker context, assembled into brief plaintext, sent to
Telegram (external service — content leaves encrypted domain at send time,
accepted tradeoff, same as Google Calendar). Brief archived to Hindsight
encrypted before retain. Obsidian file written as plaintext markdown (in
tenant's Google Drive — customer-controlled).

### Law 3: Agent Write Policy
Morning brief archival: written as `memory_type: 'episodic'`,
`provenance: 'morning_brief'`. Not a behavioral pattern — no procedural writes.
Weekly synthesis: `memory_type: 'semantic'`, `provenance: 'weekly_synthesis'`.

```typescript
// KV_SESSION key pattern for Telegram chat_id storage
// Set during onboarding (1.4 settings page or direct /start command to bot)
// kv.get(`telegram_chat_id:{tenantId}`) → chat_id string

// src/services/delivery/telegram.ts
export async function sendTelegramMessage(
  tenantId: string,
  message: string,
  env: Env,
  options?: { parseMode?: 'HTML' | 'MarkdownV2'; disablePreview?: boolean }
): Promise<boolean> {
  const chatId = await env.KV_SESSION.get(`telegram_chat_id:${tenantId}`)
  if (!chatId) return false  // Not configured — skip silently

  const res = await fetch(
    `https://api.telegram.org/bot${env.TELEGRAM_BOT_TOKEN}/sendMessage`,
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        chat_id: chatId,
        text: message,
        parse_mode: options?.parseMode ?? 'HTML',
        disable_web_page_preview: options?.disablePreview ?? true,
      })
    }
  )
  return res.ok
}
```

**Getting the chat_id:** When the tenant messages `/start` to the bot, Telegram
sends a webhook to `POST /telegram/webhook` (new route, behind Telegram IP
allowlist — not CF Access). The webhook stores the `chat_id` in KV and D1.

## Morning Brief Assembly

### Section 1 — Today's Calendar

```typescript
// Fetch today's events from Google Calendar
// Uses getGoogleToken() from 2.2 — same pattern
// Filter: today's date, all-day events + timed events
// Format: "9:00 AM — Design Review (45 min)" per event

async function fetchTodayCalendar(tenantId: string, env: Env): Promise<string> {
  const token = await getGoogleToken(tenantId, 'calendar', env)
  if (!token) return '_No calendar connected_'

  const today = new Date()
  today.setHours(0, 0, 0, 0)
  const tomorrow = new Date(today.getTime() + 86_400_000)

  const events = await fetchCalendarEvents(token, today.toISOString(), tomorrow.toISOString())
  if (!events.length) return '_No events today_'

  return events
    .sort((a, b) => a.start.localeCompare(b.start))
    .map(e => `• ${formatTime(e.start)} — ${e.summary} (${e.durationMin}m)`)
    .join('\n')
}
```

### Section 6 — News Headlines

```typescript
// Brave Search API — news endpoint
// Query: derived from tenant's active domains
// Active domains from consolidation_runs.pass5_domains (last run)
// Fallback query: "technology business finance"

async function fetchNewsHeadlines(domains: string[], env: Env): Promise<string> {
  const query = buildNewsQuery(domains)  // e.g., "cloudflare technology enterprise software"

  const res = await fetch(
    `https://api.search.brave.com/res/v1/news/search?q=${encodeURIComponent(query)}&count=5&freshness=pd`,
    { headers: { 'X-Subscription-Token': env.BRAVE_API_KEY } }
  )
  if (!res.ok) return '_News unavailable_'

  const data = await res.json() as { results: Array<{ title: string; url: string }> }
  return data.results
    .slice(0, 5)
    .map(r => `• ${r.title}`)
    .join('\n')
}
```

### Section 7 — Bible Verse

```typescript
// bible-api.com — no key required
// Random verse, KJV translation
// Cache in KV for 24h (same verse all day if brief sent multiple times)

async function fetchBibleVerse(env: Env): Promise<string> {
  const cached = await env.KV_SESSION.get('bible_verse:today')
  if (cached) return cached

  const res = await fetch('https://bible-api.com/?random=verse&translation=kjv')
  if (!res.ok) return ''

  const data = await res.json() as { reference: string; text: string }
  const verse = `📖 ${data.reference}: "${data.text.trim()}"`

  // Cache until midnight UTC
  const secondsUntilMidnight = Math.floor((86_400_000 - (Date.now() % 86_400_000)) / 1000)
  await env.KV_SESSION.put('bible_verse:today', verse, { expirationTtl: secondsUntilMidnight })

  return verse
}
```

### Brief Assembly + Delivery

```typescript
// src/cron/morning-brief.ts

async function assembleMorningBrief(tenantId: string, kek: CryptoKey, env: Env): Promise<string> {
  const [calendar, pending, highlights, gap, bridge, news, verse] = await Promise.allSettled([
    fetchTodayCalendar(tenantId, env),
    fetchPendingActions(tenantId, env),
    fetchMemoryHighlights(tenantId, kek, env),
    fetchOpenLoop(tenantId, env),
    fetchGift(tenantId, kek, env),
    fetchNewsHeadlines(await getActiveDomains(tenantId, env), env),
    fetchBibleVerse(env),
  ])

  // Promise.allSettled — each section fails independently
  const resolve = <T>(r: PromiseSettledResult<T>, fallback: T): T =>
    r.status === 'fulfilled' ? r.value : fallback

  const today = new Date().toLocaleDateString('en-US', { weekday: 'long', month: 'long', day: 'numeric' })

  return [
    `<b>Good morning — ${today}</b>`,
    '',
    `<b>📅 Today</b>`,
    resolve(calendar, '_Calendar unavailable_'),
    '',
    resolve(pending, '').length ? `<b>⏳ Pending Your Approval</b>\n${resolve(pending, '')}` : '',
    '',
    resolve(highlights, '').length ? `<b>💡 From Your Brain</b>\n${resolve(highlights, '')}` : '',
    '',
    resolve(gap, '').length ? `<b>💭 Open Loop</b>\n${resolve(gap, '')}` : '',
    '',
    resolve(bridge, '').length ? `<b>🎁 Something Interesting</b>\n${resolve(bridge, '')}` : '',
    '',
    `<b>🌍 Today's News</b>`,
    resolve(news, '_News unavailable_'),
    '',
    resolve(verse, ''),
  ].filter(s => s.length > 0).join('\n')
}

async function deliverMorningBrief(tenantId: string, brief: string, kek: CryptoKey, env: Env) {
  // 1. Telegram (primary)
  await sendTelegramMessage(tenantId, brief, env, { parseMode: 'HTML' })

  // 2. Pages UI WebSocket push
  const id = env.MCPAGENT.idFromName(tenantId)
  const stub = env.MCPAGENT.get(id)
  // @ts-expect-error DO RPC
  stub.broadcast({ type: 'brief.morning', content: brief, delivered_at: Date.now() })
    .catch(() => {})  // Non-fatal — DO may be cold

  // 3. Obsidian — write to Google Drive /from-brain/
  const driveToken = await getGoogleToken(tenantId, 'drive', env)
  if (driveToken) {
    const filename = `Daily Brief ${new Date().toISOString().split('T')[0]}.md`
    const obsidianContent = `---\ngenerated_by: the-brain\ndate: ${new Date().toISOString()}\n---\n\n${briefToMarkdown(brief)}`
    await writeToDriveBrainFolder(filename, obsidianContent, driveToken, env)
      .catch(() => {})  // Non-fatal
  }

  // 4. Archive to Hindsight as episodic memory (encrypted)
  // Build artifact from brief markdown version
  const artifact: IngestionArtifact = {
    tenantId,
    source: 'mcp_retain',
    content: briefToMarkdown(brief),
    occurredAt: Date.now(),
    memoryType: 'episodic',
    domain: 'general',
    provenance: 'morning_brief',
    metadata: { is_brief: true }
  }
  await retainContent(artifact, kek, env, ctx)

  // 5. Mark open loop gap as surfaced
  await markGapSurfaced(tenantId, env)
}
```

## Predictive Heartbeat

**30-minute cadence, 8 AM–8 PM only.** Does NOT send a message every 30 minutes —
only sends when something is actionable.

```typescript
// src/cron/heartbeat.ts (~100 lines)

async function runPredictiveHeartbeat(env: Env, ctx: ExecutionContext): Promise<void> {
  // Check time — 8 AM to 8 PM UTC only
  const hour = new Date().getUTCHours()
  if (hour < 8 || hour >= 20) return

  // For each active tenant (has open session or brief configured)
  const tenants = await env.D1_US.prepare(
    `SELECT id, hindsight_tenant_id FROM tenants
     WHERE bootstrap_status = 'completed'`
  ).all()

  await Promise.allSettled(
    tenants.results.map(t => checkTenantHeartbeat(t.id as string, env, ctx))
  )
}

async function checkTenantHeartbeat(tenantId: string, env: Env, ctx: ExecutionContext): Promise<void> {
  const alerts: string[] = []

  // Check 1: Pending action about to expire (> 20h old, still awaiting_approval)
  const expiringActions = await env.D1_US.prepare(
    `SELECT tool_name, integration, proposed_at FROM pending_actions
     WHERE tenant_id = ? AND state = 'awaiting_approval'
     AND proposed_at < ? ORDER BY proposed_at ASC LIMIT 3`
  ).bind(tenantId, Date.now() - 20 * 60 * 60 * 1000).all()

  if (expiringActions.results.length) {
    alerts.push(
      `⏰ <b>Actions expiring soon:</b>\n` +
      expiringActions.results
        .map(a => `• ${a.tool_name} via ${a.integration}`)
        .join('\n')
    )
  }

  // Check 2: High-priority consolidation gap not yet surfaced
  const unsurfacedGaps = await env.D1_US.prepare(
    `SELECT COUNT(*) as count FROM consolidation_gaps
     WHERE tenant_id = ? AND surfaced = 0 AND priority = 'high'`
  ).bind(tenantId).first<{ count: number }>()

  if ((unsurfacedGaps?.count ?? 0) > 2) {
    alerts.push(`💭 <b>${unsurfacedGaps!.count} open questions</b> from last night's analysis — review when you have a moment.`)
  }

  // Check 3: No session today (if past noon and no McpAgentDO session opened)
  // TODO: Phase 4+ — requires session tracking in D1

  // Only send if there are alerts
  if (alerts.length > 0) {
    const message = alerts.join('\n\n')
    await sendTelegramMessage(tenantId, message, env, { parseMode: 'HTML' })
  }
}
```

## Weekly Synthesis

```typescript
// Friday 5 PM UTC cron: "0 17 * * 5"
// src/cron/weekly-synthesis.ts (~100 lines)

async function runWeeklySynthesis(env: Env, ctx: ExecutionContext): Promise<void> {
  const tenants = await env.D1_US.prepare(
    `SELECT id, hindsight_tenant_id FROM tenants WHERE bootstrap_status = 'completed'`
  ).all()

  await Promise.allSettled(
    tenants.results.map(t => generateWeeklySynthesis(t.id as string, env, ctx))
  )
}

async function generateWeeklySynthesis(tenantId: string, env: Env, ctx: ExecutionContext): Promise<void> {
  const kek = await fetchAndValidateKek(tenantId, env)
  if (!kek) return  // Deferred — no KEK

  // Recall: session syntheses from last 7 days
  const weekSessions = await hindsightRecall({
    tenant_id: await getHindsightTenantId(tenantId, env),
    query_encrypted: await encryptQuery('session week review decisions', kek),
    limit: 20,
  })

  const decryptedSessions = await decryptResults(weekSessions, kek)

  // LLM synthesis
  const synthesis = await env.AI.run(
    '@cf/meta/llama-3.3-70b-instruct-fp8-fast',
    {
      messages: [{
        role: 'user',
        content: `Review this week's sessions and write a brief weekly synthesis.
Include: key themes, significant decisions, patterns noticed, one prediction for next week.
Keep it to 200 words. Be specific to what actually happened — no filler.

Sessions:
${decryptedSessions.slice(0, 10).join('\n\n---\n\n')}`
      }]
    },
    { gateway: { id: 'brain-gateway' } }
  ) as { response: string }

  const weeklyMessage = [
    `<b>📊 Week in Review</b>`,
    '',
    synthesis.response,
  ].join('\n')

  // Deliver: Telegram + Obsidian only (not Pages notification)
  await sendTelegramMessage(tenantId, weeklyMessage, env, { parseMode: 'HTML' })

  // Archive as semantic memory
  const artifact: IngestionArtifact = {
    tenantId,
    source: 'mcp_retain',
    content: synthesis.response,
    occurredAt: Date.now(),
    memoryType: 'semantic',
    domain: 'general',
    provenance: 'weekly_synthesis',
    metadata: { is_weekly_synthesis: true }
  }
  await retainContent(artifact, kek, env, ctx)
}
```

### Must Use
- `Promise.allSettled()` for brief section assembly — no section blocks others
- `{ gateway: { id: 'brain-gateway' } }` on weekly synthesis LLM call
- KV cache for Bible verse (24h TTL) — do not call bible-api.com more than once/day
- `generated_by: the-brain` frontmatter on all Obsidian output files (anti-circular)
- KEK from `kek.ts` service — do not re-implement KEK fetch inline

### Must NOT
- Send Telegram message if no chat_id configured — silent skip, not error
- Send heartbeat message when no alerts — 30-min silence is correct behavior
- Fire heartbeat outside 8 AM–8 PM UTC window
- Write brief archive to Hindsight synchronously — `ctx.waitUntil()`
- Block brief delivery on Obsidian write failure
