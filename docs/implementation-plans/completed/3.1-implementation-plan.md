### BaseAgent

#### [NEW] [base-agent.ts](file:///c:/Users/matth/Documents/HAETSAL%20OS/src/agents/base-agent.ts)

Abstract class (~160 lines). All domain agents extend this.

**Lifecycle:**
- `open()` — load mental model (semantic recall), recent episodic context (recall), pending actions (D1). Decrypt results with TMK
- `run(input)` → `agentLoop(input)` — Workers AI via AI Gateway. Token tracking from `usage.input_tokens + usage.output_tokens`. Doom loop detection per call. Context budget flush at 80%
- `close(sessionSummary)` — retain session synthesis as episodic memory via `retainContent()` (ctx = `undefined` in DO). Encrypt reasoning trace → R2_OBSERVABILITY. Clear session state

**Key methods:**
- `retain(content, memoryType: EpistemicMemoryType, domain, provenance)` — the **ONLY** memory write path. Type-level enforcement: `'procedural'` won't compile
- `hindsightRecall(req)` — wrapper calling `env.HINDSIGHT.fetch(...)` with encrypted query
- `primeContext(results, tmk)` — decrypt recall results, populate context
- `computeInputHash(toolName, input)` — SHA-256 for doom loop detection
- `checkDoomLoop(state, toolName, input)` — 3 warn / 5 break
- `shouldFlush(cumulativeTokens)` — `cumulativeTokens / MODEL_CONTEXT_LIMIT > 0.8`

**Constants:**
- `MODEL_CONTEXT_LIMIT = 128_000` (llama-3.3-70b-instruct-fp8-fast)
- `FLUSH_THRESHOLD = 0.80`

**Doom loop warn injection message:**
```
"[SYSTEM: You have called {toolName} with identical inputs {n} times. Consider a different approach or ask the user for clarification.]"
```

**Circuit break behavior:** Hard stop. Write anomaly signal to `anomaly_signals`. Return user-friendly message. Do NOT continue the agent loop.

### Layer 1 Router

#### [NEW] [router.ts](file:///c:/Users/matth/Documents/HAETSAL%20OS/src/services/agents/router.ts)

Pattern-first, classifier fallback (~80 lines):

- `ROUTING_PATTERNS` — array of `{ pattern: RegExp, agent: AgentType }`:
  - career/work/job/project → `career_coach`
  - workout/health/sleep → `life_coach`
  - budget/invoice/expense → `career_coach`
  - relationship/friend/family → `chief_of_staff`
  - remind/schedule/calendar → `chief_of_staff`
- `routeRequest(input, env): Promise<AgentType>` — pattern check first (~5ms), fallback to Workers AI 8B classifier via AI Gateway (~200ms)
- Classifier prompt: classify into one of 4 categories. Answer with one word. Input truncated to 300 chars
- Default fallback: `'chief_of_staff'`
- All Workers AI calls use `{ gateway: { id: 'brain-gateway' } }`
